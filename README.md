# Controle Digital - IFPR

Repositório de atividades práticas da disciplina de Controle Digital.

## Como acessar o ambiente de desenvolvimento

O ambiente roda no navegador, sem necessidade de instalar nada no computador. Siga os passos abaixo.

### 1. Criar uma conta no GitHub

Se ainda não tem conta, acesse [github.com](https://github.com) e clique em **Sign up**. Use seu e-mail institucional se possível.

### 2. (Recomendado) Solicitar o Student Developer Pack

O GitHub oferece benefícios gratuitos para estudantes, incluindo mais horas de uso do ambiente na nuvem. Acesse [education.github.com/pack](https://education.github.com/pack) e clique em **Get your Pack**. Será necessário comprovar que você é estudante (e-mail institucional ou documento de matrícula). A aprovação pode levar alguns dias.

### 3. Abrir o ambiente de desenvolvimento

1. Acesse o repositório: [github.com/WilhelmMeyer/IFPR-Controle-Digital](https://github.com/WilhelmMeyer/IFPR-Controle-Digital)
2. Clique no botão verde **Code**
3. Selecione a aba **Codespaces**
4. Clique em **Create codespace on main**
5. Aguarde a configuração do ambiente (a primeira vez pode levar 2 a 3 minutos)

Quando o ambiente terminar de carregar, o VS Code abrirá no navegador com todas as bibliotecas já instaladas.

### 4. Abrir um notebook

Na barra lateral esquerda (Explorer), navegue até a pasta `notebooks/` e clique no arquivo `.ipynb` da atividade desejada. O notebook abrirá no editor Jupyter integrado.

Para executar uma célula de código, clique nela e pressione **Shift + Enter**.

### 5. Salvar e baixar o trabalho

Para baixar o notebook resolvido: clique com o botão direito no arquivo na barra lateral e escolha **Download**. Entregue o arquivo conforme orientação do professor.

### 6. Encerrar o Codespace

Ao terminar, é importante parar o Codespace para economizar suas horas gratuitas.

**Opção 1:** No canto inferior esquerdo do VS Code, clique em **Codespaces** e selecione **Stop Current Codespace**.

**Opção 2:** Acesse [github.com/codespaces](https://github.com/codespaces), localize seu Codespace na lista, clique nos três pontos (**...**) e escolha **Stop codespace**.

O Codespace também para automaticamente após 30 minutos de inatividade.

## Informações sobre o ambiente

O Codespace já vem configurado com as seguintes ferramentas:

- Python 3.12
- JupyterLab
- NumPy, SciPy, Matplotlib
- SymPy
- lcapy
- python-control
